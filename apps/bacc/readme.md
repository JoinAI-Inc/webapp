# BACC - Bridge Across Chinese Culture

这是一个旨在帮助非中国人更好地参与和体验中国春节庆祝活动的 AI 驱动 Web 应用。通过先进的 AI 生成技术，让全球用户都能沉浸式体验中国传统文化。

---

## 🌟 核心功能

### 1. 汉服形象生成 (Hanfu Portrait)
*   **智能变装**：用户上传个人照片，系统通过 AI 自动将其转化为身着精致汉服的形象。
*   **深度自定义**：支持自定义汉服的样式、颜色、花纹及配饰，满足个性化审美。
*   **社交传播**：一键生成高清海报，支持分享至 Instagram, TikTok, WeChat 等社交平台。

### 2. 拜年视频定制 (CNY Greeting Video)
*   **动态体验**：用户上传照片后，系统生成一段自然生动的拜年视频（如拱手礼、作揖等内。
*   **形象同步**：视频中的人物形象可同步应用用户自定义的汉服样式。
*   **个性化定制**：支持自定义背景音乐、祝福语语言（中/英/法等多语言可选）及背景视觉风格。
*   **社交分享**：支持导出短视频格式，方便社交媒体传播。

### 3. 家装过年化 (Smart Home Decor)
*   **环境转换**：用户上传自家或任何建筑的图片，系统通过 AI 自动添加春联、灯笼、中国结等春节核心视觉元素。
*   **创意自定义**：用户可自选装饰风格（如：大红大紫盛唐风、极简新中式等）和配色方案。
*   **互动展示**：生成前后对比效果，展示独特的中国年氛围。

## 🏗️ 技术架构

### 系统架构图
该应用采用典型的 **下一代 Web 应用架构**，确保高并发下的生成体验和数据安全。

*   **前端 (Frontend)**: 基于 **Next.js 14+** 的响应式设计，提供极致的渲染速度和 SEO 优化。
*   **后端 (Backend)**: **Node.js/TypeScript** 异步任务驱动架构，处理复杂的 AI 生成请求。
*   **AI 引擎层**: 
    - **图像生成**: Stable Diffusion / LoRA (汉服与装饰细节) + InsightFace (人脸融合)。
    - **视频生成**: 基于预训练视频模型 (如 Kling/Luma API) 的动作驱动生成。
*   **数据流**: 采用 **Redis 队列** 管理大模型计算任务，通过 **WebSocket** 实时向前端推送生成进度。

## 🤖 Work Agent 协作分工

本项目采用多 Agent 协作模式进行高效开发，各 Agent 分工如下：

### 1. 🎨 UI/UX 视觉大师 (DesignGPT)
*   **职责**：定义项目的视觉基调（CNY Premium 风格），设计高保真 UI 组件，优化交互动效。
*   **核心关注**：色彩和谐度（深红/金）、玻璃拟态效果、响应式布局的精致感。

### 2. ⚛️ 前端工程专家 (FrontGPT)
*   **职责**：基于 Next.js 14 实现业务逻辑，集成 Framer Motion 动效，确保极致的前端性能。
*   **核心关注**：App Router 架构、客户端状态管理、极速的首屏加载体验。

### 3. 🧠 AI 流程科学家 (AIGPT)
*   **职责**：设计图像/视频生成的工作流（ComfyUI/SD LoRA），调优 Prompt，集成 AI 生成接口。
*   **核心关注**：人脸融合的自然度、汉服样式的细节还原、视频生成的连贯性。

### 4. ⚙️ 后端及架构专家 (BackGPT)
*   **职责**：构建高并发的异步 API，管理 Redis 任务队列，设计数据库架构与认证流程。
*   **核心关注**：API 稳定性、生成任务的削峰填谷、数据存储的安全与隐私。
